## Description

Our assigned disability is muscular dystrophy, a condition that causes progressive muscle weakness and makes movement increasingly difficult over time. Our game, Let Me Out, is a sliding block puzzle where players must move the yellow target block to the exit on the right side of the grid by sliding other blocks out of the way.

The core mechanics focus on three key elements. First, block movement is restricted as each block can only slide in one direction based on its orientation, requiring players to think strategically about how to rearrange the grid. Second, the heaviness of the blocks is reflected through slower reaction time, making each action feel more difficult and slow as levels progress. Finally, a move counter tracks how many actions the player takes, emphasizing efficiency and reinforcing the increasing difficulty of completing simple tasks.

## Setup and Interaction Instructions

Let Me Out can be played either on touch screen or using the arrow keys, they cannot be used at the same time. The player will be welcomed with the home screen with the option to press play which starts at the tutorial, or click the levels button to choose which level to start from. Each level will have a limited amount of moves but the player can play as many times as they want no matter if fail or pass.

## Learning Goals

- Understand how sliding block puzzle mechanics and movement constraints influence problem solving
- Practice implementing grid-based movement where blocks move only in one direction based on orientation
- Learn how increasing level constraints and slower movement can raise game difficulty
- Explore how game mechanics can simulate the growing effort required for movement in muscular dystrophy
